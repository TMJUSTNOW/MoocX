| dämmern   | to dawn              |
| donnern   | to thunder           |
| dunkeln   | to get dark          |
| frieren   | to freeze            |
| gewittern | to storm             |
| hageln    | to hail              |
| nieseln   | to drizzle           |
| gießen    | to flood/downpour    |
| regnen    | to rain              |
| schneien  | to snow              |
| tauen     | to melt              |
