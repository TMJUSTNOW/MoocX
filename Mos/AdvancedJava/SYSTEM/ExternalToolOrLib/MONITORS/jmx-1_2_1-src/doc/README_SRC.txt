---------------------------------------------------------------------------
       Java(TM) Management Extensions (JMX(TM)) v1.2.1 sources README
---------------------------------------------------------------------------


This README file describes how to build the JMX 1.2.1 sources.


The JMX 1.2.1 source delivery comes with a build.xml file written for Ant 1.5.
Please download and install Apache Ant 1.5 from: http://ant.apache.org

Once you have set your PATH, ANT_HOME and JAVA_HOME variables as described
in the Ant 1.5 installation procedure, you can build the JMX 1.2.1 sources:
* cd to the directory containing the build.xml file
* type: ant all
